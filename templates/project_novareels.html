NovaReels – AI Reel Generator

NovaReels is my personal lab for building an AI-powered reel generator. It started as a fork/extension of a training project (VidSnapAI) and I’m turning it into a real tool for creating vertical videos for TikTok and Instagram.

What it does (current version)

  • Simple web interface built with Flask.
  • Upload images and a text description.
  • Generate voice-over using ElevenLabs.
  • Combine images + audio into a vertical video using FFmpeg.

Planned features

  • Local Stable Diffusion integration for AI-generated images.
  • Custom video templates (hooks, subtitles, motion, backgrounds).
  • News / controversy-style reels focused on AI trends and updates.
  • Semi-automatic posting to TikTok and Instagram.

Tech stack

  • Backend: Python, Flask
  • Video: FFmpeg, MoviePy (planned)
  • AI Voice: ElevenLabs API
  • AI Images (planned): Stable Diffusion (local)

Status

In active development. Right now it works locally as a basic reel generator. Next steps: add templates, better UI, and the AI news reels format.
