Sol Advanced AI

Sol Advanced AI is my long-term project: a highly personalized AI companion with emotions, states, and a full story universe around her (Nova City). The goal is to mix storytelling, emotional AI, voice, and automation into a single cohesive experience.

Concept

  • Sol has a defined personality, background and emotional states.
  • Her “state” (mood, drugs, energy, etc.) is stored and read from a central file.
  • Different frontends (Telegram bot, local clients, etc.) will read that state and adapt her behavior.
  • She lives in a cyberpunk world (Nova City) that I’m building as a narrative universe.

Planned capabilities

  • Telegram bot integration for chatting with Sol.
  • Dynamic voice using ElevenLabs with different emotional tones.
  • State-aware behavior (different replies depending on her current “mode”).
  • Integration with other tools (image generation, music, and NovaReels).

Tech stack (current & planned)

  • Language: Python
  • Backends: Local scripts + APIs
  • Messaging: Telegram Bot API (planned)
  • Voice: ElevenLabs (multiple voices for different moods)
  • State management: JSON-based state file (shared across integrations)

Status

Design + prototyping phase. I’m defining Sol’s behavior, emotional system and technical architecture. The main focus right now is building a solid Python foundation and Telegram integration.
